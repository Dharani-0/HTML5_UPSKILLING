# HTML5 Upskilling Exercises

This repository contains HTML5 practice exercises completed as part of the CTS Upskilling Program.

## Exercises Included

1. HTML5 Base Template
2. Navigation and Linking
3. Welcome Message with Styling and ID/Class
4. Image Gallery for Community Events
5. Image Handling
6. Event Handling
7. Geolocation
8. Debugging HTML
9. Saving Data
10. Video Embedding

## Technologies Used

- HTML5
- CSS3

## Purpose

The purpose of this repository is to practice and demonstrate HTML5 concepts such as semantic elements, navigation, forms, multimedia, event handling, geolocation, and basic styling.
